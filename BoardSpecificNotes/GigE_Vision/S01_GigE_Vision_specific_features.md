# Zebra GigE Vision driver overview

The Zebra GigE Vision driver allows you to use a Gigabit Ethernet (1000 BaseT or faster) network adapter to grab from GigE Vision-compliant cameras with Aurora Imaging Library. The Gigabit Ethernet network adapter can be a third-party device, or a Zebra product (such as the Zebra Concord PoE).
