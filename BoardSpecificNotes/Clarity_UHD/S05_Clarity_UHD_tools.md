# Zebra Clarity UHD tools

There is one Zebra Clarity UHD external application (utility) available (Zebra Clarity UHD Bench) and one Zebra Clarity UHD tool integrated in the Aurora Imaging Configurator utility (Zebra Clarity UHD System Monitor tool). To access these utilities and this tool, launch the Aurora Imaging Configurator utility. Then, select the **Boards Clarity UHD** item from the tree structure in the presented interface. The utilities are located in the **Launch external applications** and **Zebra Clarity UHD System Monitor** panes, respectively.

## Zebra Clarity UHD Bench utility

The Zebra Clarity UHD Bench utility calculates the real-time transfer speed of the PCIe slot (in Mbytes/sec) from:

- Zebra Clarity UHD to Host.
- Host to Zebra Clarity UHD.

## Zebra Clarity UHD System Monitor tool

The Zebra Clarity UHD system monitor tool is integrated in the Aurora Imaging Configurator utility. It is used to display the temperature of the acquisition controller and the Host interface of the board.
