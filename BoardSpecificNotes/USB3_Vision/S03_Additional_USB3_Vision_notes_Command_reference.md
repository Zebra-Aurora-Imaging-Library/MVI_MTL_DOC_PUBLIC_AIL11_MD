# Miscellaneous reference information for Zebra USB3 Vision

This section includes additional command reference information for Zebra USB3 Vision. The information found in this section might be a reiteration of content previously documented.

## Mbuf

This section presents information about Zebra USB3 Vision and the buffer module.

### MbufAlloc...

The following lists additional USB3 Vision support when using [`MbufAlloc...`](../../Reference/buf/MbufAlloc1d.md).

- Support for **M_IMAGE** + **M_GRAB** + **M_DYNAMIC** buffer attribute.
