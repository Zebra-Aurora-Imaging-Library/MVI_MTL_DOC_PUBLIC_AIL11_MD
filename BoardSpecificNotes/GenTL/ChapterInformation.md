# Zebra GenTL Consumer

This chapter discusses features of Aurora Imaging Library that are particular to the Zebra driver for GenTL (Aurora Imaging Library GenTL Consumer), and optimizing its performance. Information regarding using this driver with specific functions is integrated into the functions' reference topics. The Aurora Imaging Library release notes, accessible from Aurora Imaging Control Center and/or the Zebra Aurora Imaging Library product support webpages (for example, [zebra.com/ail-info](https://zebra.com/ail-info)), can provide additional documentation, such as new or modified features, systems, and products.
