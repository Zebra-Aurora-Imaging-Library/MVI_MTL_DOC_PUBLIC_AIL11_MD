# Code grading examples

The _CodeGrading.cpp_ example shows a code grading operation run on several types of linear, composite, and cross-row codes.

The _ISODPMGrading.cpp_ example illustrates how to grade a target image containing a DPM Data Matrix code type. The example's grading operation and results are based on ISO/IEC 29158 quality guidelines.

To run/view these and other examples, use Aurora Imaging Example Launcher.
