# Geometric Model Finder example

The Model Finder example, _MMod.cpp_ illustrates how the model can be used with the following source images:

*[Image: ExampleImages.png]*

Specifically, _MMod.cpp_ shows how to define a Model Finder context with a single model, as well as with several models, and find these models in a target image.

> **Code example:** [MMod.cpp](MMod.cpp)

To run this example, use the Aurora Imaging Example Launcher in the Aurora Imaging Control Center.
