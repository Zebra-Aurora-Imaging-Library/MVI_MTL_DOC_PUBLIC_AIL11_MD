# SureDotOCR example

The SureDotOCR example _Mdmr.cpp_ illustrates how to read dot-matrix text. Specifically, the example reads a product's expiry date and lot number.

*[Image: MdmrExampleMain.png]*

> **Code example:** [Mdmr.cpp](Mdmr.cpp)

To run this example, use the Aurora Imaging Example Launcher in the Aurora Imaging Control Center.
