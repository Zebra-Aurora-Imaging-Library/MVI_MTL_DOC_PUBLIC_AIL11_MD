# Bead example

The bead example _mbead.cpp_ shows you how to train a bead template using a training image, and how to use the trained template to verify a bead for defects in a target image. The example deals with bead paths that follow a polyline and a fixed circle.

*[Image: BeadTrainingAndTarget.png]*

> **Code example:** [mbead.cpp](mbead.cpp)

To run this example, use the Aurora Imaging Example Launcher in the Aurora Imaging Control Center.
