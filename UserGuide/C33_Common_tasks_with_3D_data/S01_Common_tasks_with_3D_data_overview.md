# Common tasks with 3D data overview

You can perform a variety of tasks by processing and analyzing 3D data using the Aurora Imaging Library 3D modules.

Note that different 3D modules can be used to perform the same task. For example, you can find an object using the 3D Blob Analysis module, the 3D Model Finder module, the 3D Measurement module, the 3D Metrology module, or the 3D Registration module. However, to obtain optimal results, you should choose the 3D module that best suits your specific needs. For example, you should use the 3D Model Finder module when you want to determine the location and orientation of multiple rectangular planes.

This chapter discusses typical 3D use cases, including the suggested steps to take and 3D modules to use to in an attempt to obtain optimal results, depending on the requirements of your application.
