# Industrial communication overview

The Aurora Imaging Library Industrial Communication module allows you to communicate information to other devices, such as robots, programmable logic controllers (PLCs), or I/O controllers. The Industrial Communication module allows you to send and receive data between such devices. For example, in the typical case for applications that communicate with robots, you would wait to receive the robot's current location, and then send a new location for it to move to.
