# Optical character recognition example

The Optical Character Recognition example, _MOcr.cpp_ illustrates how the module can be used with the following source image:

*[Image: ocrsemi1.png]*

Specifically, _MOcr.cpp_ shows how to read the serial number in an image of a semiconductor wafer, using the OCR functions in conjunction with other Aurora Imaging Library functions. The serial number, printed on the wafer, is a standard SEMI M12 92 font character string, containing a checksum.

> **Code example:** [mocr.cpp](mocr.cpp)

To run this example, use the Aurora Imaging Example Launcher in the Aurora Imaging Control Center.
