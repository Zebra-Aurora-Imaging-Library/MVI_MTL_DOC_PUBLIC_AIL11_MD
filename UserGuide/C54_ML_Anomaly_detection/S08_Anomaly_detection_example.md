# Anomaly detection example

The example _ClassAnomalyDetectionCompleteTrain.cpp_ demonstrates how to train a predefined (Zebra defined) anomaly detection classifier context to detect defects on the top of glass bottles. This example also shows you how to evaluate the performance of the trained classifier on the test dataset, using [`MclassStatCalculate`](../../Reference/class/MclassStatCalculate.md).

*[Image: MclassAnomalyDetectionExampleOfficial.png]*

To run/view this and other examples, use Aurora Imaging Example Launcher.
