# Distribution of Aurora Imaging Library applications

There are some details that you must consider before you can distribute an Aurora Imaging Library or Aurora Imaging Library Lite application, either for your customers' use or for your own. When doing this distribution, you must redistribute Aurora Imaging Library/Aurora Imaging Library Lite DLLs and device drivers. In addition, when using an Aurora Imaging Library processing module, you must purchase an appropriate runtime license; otherwise, your Aurora Imaging Library application will not function correctly. Aurora Imaging Library does not automatically install a license. The user must install the license, whether it is a provisional, development, or runtime license.

This chapter deals with the different ways to distribute your application, obtain a license, and handle licensing for your customers.

> **Note:** Note that Aurora Imaging Library Lite applications will work if a Zebra Imaging board or a supplemental license is present. An Aurora Imaging Library installation without any license will work as an Aurora Imaging Library Lite runtime installation if a Zebra Imaging board is present.
