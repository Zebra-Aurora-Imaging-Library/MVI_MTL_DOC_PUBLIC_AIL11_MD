# Dongles for activating runtime and development licenses

Dongles are used to activate Aurora Imaging Library development and runtime licenses.

If you decide that another computer needs to run an Aurora Imaging Library application, you can transfer the dongle. This will allow the second computer to use Aurora Imaging Library, but the first computer will have Aurora Imaging Library disabled.

Note that you can use the same dongle with Aurora Imaging Library X and 11.

To have Aurora Imaging Library DLLs recognize the dongle, you must install the CodeMeter Runtime Kit software. To install this software, if you did not install it when you were installing Aurora Imaging Library, run the _CodemeterRuntime64.exe_ file, located in the _\Install\CmStick_ directory of the Aurora Imaging Library installation media, and follow the on-screen instructions. Note, the software is installed automatically whenever you install Aurora Imaging Library, but might not be installed if Aurora Imaging Library was redistributed to your computer using _Redist.exe_.
