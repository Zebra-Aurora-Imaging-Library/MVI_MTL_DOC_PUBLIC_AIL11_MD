# Welcome to Aurora Imaging Library

Aurora Imaging Library is a state-of-the-art machine vision software development kit (SDK) with a deep collection of optimized functions and tools. With Aurora Imaging Library, you can tackle any machine vision task, from simple image capture, to code reading, string reading, deep learning, and 2D/3D matching.

*[Image: Main.png]*

## What's new

This guide was last published: 2025-12-01.

Aurora Imaging Library 11 offers new and improved features in virtually every module and aspect of the SDK, and now lets you work with side-by-side installations of version 10.7 and version 11 on the same computer. Version 11 includes an overall product rebranding consistent with Zebra machine vision products. For more information, [click here](S02_Whats_new.md).

## Going to Aurora Imaging Library 11 from 10.7

Users going to Aurora Imaging Library 11 from Aurora Imaging Library 10.7 will notice many improvements and modifications, such as the rebranding of tools, the ability to use 11 and 10.7 side-by-side, and a conversion utility to help update source code to 11 from 10.7. For more information, [click here](../C02_Building_an_application/S03_Going_from_AIL_10.7_to_AIL_11.md).

## Aurora Imaging Object Library

Aurora Imaging Object Library is a new alternative object-oriented API to the classic Aurora Imaging Library API. For more information, [click here](../../AIOL/index.md).

## Updates and release notes

Aurora Imaging Library updates, such as new or modified systems, drivers, features, tools, and add-ons, are available for download. Be sure to check for these, as they can expand the library's capabilities, and improve the development and execution of your applications. For more information, [click here](../../ReleaseNotes/index.md).

> **Note:** The Aurora Imaging Library release notes, accessible from Aurora Imaging Control Center and/or the Zebra Aurora Imaging Library product support webpages (for example, [zebra.com/ail-info](https://zebra.com/ail-info)), can provide additional documentation, such as new or modified features, systems, and products. You can also access the latest version of the Help and other related resources at [zebra.com/ail-docs](https://zebra.com/ail-docs).

For the latest information, always consult the online Help, which is more current than the installed version.

## Tools and utilities

A wide range of tools and utilities are available to you, starting with Aurora Imaging Control Center, which is a user interface that serves as the library's main hub. From there, you can access all important tools, utilities, and files, such as Aurora Imaging Configurator, Aurora Imaging CoPilot, Aurora Imaging Example Launcher, and Aurora Imaging Intellicam. For more information, [click here](../C01_Tools_and_utilities/ChapterInformation.md).
