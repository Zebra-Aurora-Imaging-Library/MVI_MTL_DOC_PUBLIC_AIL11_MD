# Calculating distance measurements and statistics

The 3D Metrology module can calculate distance measurements between a source and reference Aurora Imaging Library object, and compute a variety of statistics (such as, the minimum, maximum and average distances) on those distance measurements.

You can calculate distance measurements between two Aurora Imaging Library objects, using [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md) or [`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md). You can also calculate distances and perform statistics, using [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md). Note that [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md) uses a distance 3D metrology context and stores the result of the distance operation in a result buffer. [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md) uses a statistics 3D metrology context and stores the result of the statistics operation in a result buffer. [`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md) does not require a context or result buffer and returns the calculated distance directly.

> **Note:** To calculate distances and statistics for a single point cloud or depth map (such as, the bounding box or the distances between the points), use the 3D image processing module ([`M3dimStat`](../../Reference/3dim/M3dimStat.md)) instead. For more information, see [Calculating statistics on a point cloud, depth map, or 3D geometry](../C35_3D_Image_processing/S13_Calculating_statistics_on_a_point_cloud_or_depth_map.md).

## Steps to calculate distance measurements and statistics

The following steps provide a basic methodology for calculating distance measurements and statistics:

1. Optionally, allocate a statistics 3D metrology context, using [`M3dmetAlloc`](../../Reference/3dmet/M3dmetAlloc.md) with [`M_STATISTICS_CONTEXT`](../../Reference/3dmet/M3dmetAlloc.md) or, allocate a distance 3D metrology context with [`M_DISTANCE_CONTEXT`](../../Reference/3dmet/M3dmetAlloc.md).
   > **Note:** Note that if you want to evaluate a single statistics calculation, you can call [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md) with a predefined statistics 3D metrology context, without having to allocate a statistics 3D metrology context beforehand.
2. Allocate a statistics 3D metrology result buffer, using [`M3dmetAllocResult`](../../Reference/3dmet/M3dmetAllocResult.md) with [`M_STATISTICS_RESULT`](../../Reference/3dmet/M3dmetAllocResult.md) or, allocate a calculate 3D metrology result buffer with [`M_CALCULATE_RESULT`](../../Reference/3dmet/M3dmetAllocResult.md).
3. Control the settings of the statistics or distance 3D metrology context using [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md). These settings determine which distance measurements and statistics to calculate.
4. Call [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md) to calculate the distance measurements and evaluate their statistics, or call [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md) to calculate the distance measurements without evaluating their statistics.
5. Retrieve results from the statistics or distance 3D metrology result buffer, using [`M3dmetGetResult`](../../Reference/3dmet/M3dmetGetResult.md).

Note that you can calculate distance measurements without evaluating statistics or allocating a 3D metrology context or result buffer using [`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md). This function creates a distance map which is useful, for example, to highlight defects using a LUT in a 2D or 3D display. For more information about showing point clouds in a 3D display using a LUT, see [Applying LUTs](../C43_3D_Display_and_graphics/S04_Color_and_display_settings_for_3D_data.md). You must use [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md) if you want to save the paired reference facets, pixels, and points ([`M_SAVE_PAIRED_REFERENCE_...`](../../Reference/3dmet/M3dmetControl.md)).

## How distances are calculated

Depending on the type of reference object that you pass, and the setting of the [`DistanceType`](../../Reference/3dmet/M3dmetStat.md) parameter, [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md) and [`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md) can both calculate different distances for each point in the source. You must specify the distance type to be calculated, even if only one option is available (for example, when the reference is a point cloud without a mesh). To use [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), allocate a distance 3D metrology context ([`M3dmetAlloc`](../../Reference/3dmet/M3dmetAlloc.md)) and use [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_PAIRING_TYPE`](../../Reference/3dmet/M3dmetControl.md), [`M_PAIRING_DISTANCE_METRIC`](../../Reference/3dmet/M3dmetControl.md), and [`M_DISTANCE_METRIC_OPERATOR`](../../Reference/3dmet/M3dmetControl.md) to specify the distance to calculate. Alternatively, use [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md) with a predefined distance 3D metrology context ([`M_DISTANCE_CONTEXT_...`](../../Reference/3dmet/M3dmetDistanceEx.md)).

> **Note:** When a depth map is passed as a source to these functions, it is typically internally converted to a point cloud. When referring to the source object in this section, the term point can refer to either a point in a point cloud, or the position in 3D space represented by a pixel of a depth map. This does not apply to the reference object; when passed as a reference, depth maps and point clouds are treated differently.

Only some combinations of reference object, [`M_PAIRING_TYPE`](../../Reference/3dmet/M3dmetControl.md), [`M_PAIRING_DISTANCE_METRIC`](../../Reference/3dmet/M3dmetControl.md), and [`M_DISTANCE_METRIC_OPERATOR`](../../Reference/3dmet/M3dmetControl.md) are accepted when using [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md). The following table lists all of the possible combinations of these values:

|   |   |   |   |
| --- | --- | --- | --- |
| [`M_PAIRING_TYPE`](../../Reference/3dmet/M3dmetControl.md) | Reference object | [`M_PAIRING_DISTANCE_METRIC`](../../Reference/3dmet/M3dmetControl.md) | [`M_DISTANCE_METRIC_OPERATOR`](../../Reference/3dmet/M3dmetControl.md) |
| [`M_DISTANCE_TO_REFERENCE`](../../Reference/3dmet/M3dmetControl.md) | 3D point, 3D sphere, 3D cylinder, or 3D line geometry | [`M_EUCLIDEAN`](../../Reference/3dmet/M3dmetControl.md) | [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SQUARED`](../../Reference/3dmet/M3dmetControl.md) |
| [`M_DISTANCE_TO_SURFACE`](../../Reference/3dmet/M3dmetControl.md) | 3D sphere geometry | [`M_EUCLIDEAN`](../../Reference/3dmet/M3dmetControl.md) | [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SIGNED`](../../Reference/3dmet/M3dmetControl.md) |
| 3D plane geometry | [`M_EUCLIDEAN`](../../Reference/3dmet/M3dmetControl.md) | [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SQUARED`](../../Reference/3dmet/M3dmetControl.md) |
| 3D plane geometry | [`M_DIRECTION_Z`](../../Reference/3dmet/M3dmetControl.md) | [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SIGNED`](../../Reference/3dmet/M3dmetControl.md) |
| 3D box geometry | [`M_EUCLIDEAN`](../../Reference/3dmet/M3dmetControl.md) | [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SIGNED`](../../Reference/3dmet/M3dmetControl.md) |
| 3D box geometry | [`M_MANHATTAN`](../../Reference/3dmet/M3dmetControl.md) | [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) |
| 3D cylinder | [`M_EUCLIDEAN`](../../Reference/3dmet/M3dmetControl.md) | [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SIGNED`](../../Reference/3dmet/M3dmetControl.md) |
| Meshed point cloud | [`M_EUCLIDEAN`](../../Reference/3dmet/M3dmetControl.md) or [`M_DIRECTION_NORMAL`](../../Reference/3dmet/M3dmetControl.md) | [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) |
| [`M_DISTANCE_TO_NEIGHBOR`](../../Reference/3dmet/M3dmetControl.md) | Point cloud | [`M_EUCLIDEAN`](../../Reference/3dmet/M3dmetControl.md) or [`M_DIRECTION_NORMAL`](../../Reference/3dmet/M3dmetControl.md) | [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) |
| Depth map | [`M_DIRECTION_Z`](../../Reference/3dmet/M3dmetControl.md) | [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SIGNED`](../../Reference/3dmet/M3dmetControl.md) |

In the case that only one setting is available, and you specify an unavailable setting, it will be ignored and the appropriate setting will be used instead. When more than one setting is available, and an unavailable setting is specified, an error is generated.

### Point cloud as a reference

When the reference is a point cloud without a mesh, distances are measured from each point in the source to the nearest point in the reference. To calculate these distances using [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md) or[`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md), set[`DistanceType`](../../Reference/3dmet/M3dmetStat.md) to [`M_DISTANCE_TO_NEAREST_NEIGHBOR`](../../Reference/3dmet/M3dmetStat.md). When using [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), use [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_PAIRING_TYPE`](../../Reference/3dmet/M3dmetControl.md) set to [`M_DISTANCE_TO_NEIGHBOR`](../../Reference/3dmet/M3dmetControl.md) for the same result. You can also find the nearest neighbor of the source points in the direction of their normals with [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), using [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with[`M_PAIRING_DISTANCE_METRIC`](../../Reference/3dmet/M3dmetControl.md) set to [`M_DIRECTION_NORMAL`](../../Reference/3dmet/M3dmetControl.md). To compute the normal distance between the paired source and reference points, you must enable [`M_DISTANCE_NORMAL`](../../Reference/3dmet/M3dmetControl.md). The normal distance is defined as the dot product between the paired source point's normal and the reference point's normal.

When the reference is a meshed point cloud, you can choose to instead measure distances from each point in the source to the nearest position on the surface of the mesh. To calculate these distances using [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md) or[`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md), set[`DistanceType`](../../Reference/3dmet/M3dmetStat.md) to [`M_DISTANCE_TO_MESH`](../../Reference/3dmet/M3dmetStat.md). When using [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), use [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_PAIRING_TYPE`](../../Reference/3dmet/M3dmetControl.md) set to [`M_DISTANCE_TO_SURFACE`](../../Reference/3dmet/M3dmetControl.md) for the same result. You can also find the nearest position on the surface of the mesh to the source points in the direction of their normals with [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), using [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_PAIRING_DISTANCE_METRIC`](../../Reference/3dmet/M3dmetControl.md) set to [`M_DIRECTION_NORMAL`](../../Reference/3dmet/M3dmetControl.md).

> **Note:** For maximum performance, you can specify not to complete calculations for source points that are more than a certain distance from the reference point cloud. For [`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md), specify this distance using the [`Param`](../../Reference/3dmet/M3dmetDistance.md)parameter. For [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), specify this distance using [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_PAIRING_DISTANCE_METRIC`](../../Reference/3dmet/M3dmetControl.md) and [`M_PAIRING_DISTANCE_MAX`](../../Reference/3dmet/M3dmetControl.md). In the case that the reference object is a point cloud and [`M_PAIRING_TYPE`](../../Reference/3dmet/M3dmetControl.md) is set to [`M_DISTANCE_TO_NEIGHBOR`](../../Reference/3dmet/M3dmetControl.md), use [`M_PAIRING_PERPENDICULAR_DISTANCE_MAX`](../../Reference/3dmet/M3dmetControl.md) and [`M_PAIRING_PERPENDICULAR_DISTANCE_MAX_MODE`](../../Reference/3dmet/M3dmetControl.md). For [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md), specify this distance using the [`Condition`](../../Reference/3dmet/M3dmetStat.md), [`CondHigh`](../../Reference/3dmet/M3dmetStat.md)and [`CondLow`](../../Reference/3dmet/M3dmetStat.md)parameters, which also determine whether a given distance is used when calculating statistics.

### Depth map as a reference

When the reference is a fully corrected depth map, distances are measured from each point in the source to the position in the depth map that is directly above or below the source point along the Z-axis. To calculate these distances, using [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md) or[`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md), set[`DistanceType`](../../Reference/3dmet/M3dmetStat.md) to[`M_ABSOLUTE_DISTANCE_Z_TO_SURFACE`](../../Reference/3dmet/M3dmetStat.md)or [`M_SIGNED_DISTANCE_Z_TO_SURFACE`](../../Reference/3dmet/M3dmetStat.md). For the same result when using [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), use [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_DISTANCE_METRIC_OPERATOR`](../../Reference/3dmet/M3dmetControl.md) set to [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SIGNED`](../../Reference/3dmet/M3dmetControl.md).

> **Note:** If a point does not map to a position in the depth map, it is assigned a distance of**AIL_FLOAT_MAX**and is ignored for statistics calculations.

### 3D geometry as a reference

When the reference is a 3D geometry object, you can choose to measure distances in several ways depending on the type of the 3D geometry:

- For all types of 3D geometry objects (except lines), you can choose to measure distances from each point to the nearest position on the surface of the 3D geometry. To calculate these distances using [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md) or[`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md), set[`DistanceType`](../../Reference/3dmet/M3dmetStat.md) to [`M_ABSOLUTE_DISTANCE_TO_SURFACE`](../../Reference/3dmet/M3dmetStat.md), [`M_SIGNED_DISTANCE_TO_SURFACE`](../../Reference/3dmet/M3dmetStat.md), or [`M_MANHATTAN_DISTANCE_TO_SURFACE`](../../Reference/3dmet/M3dmetStat.md)(for a 3D box geometry only). For [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), use [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_PAIRING_TYPE`](../../Reference/3dmet/M3dmetControl.md) set to [`M_DISTANCE_TO_SURFACE`](../../Reference/3dmet/M3dmetControl.md) for the same result. To compute the normal distance between the paired source and reference points, you must enable [`M_DISTANCE_NORMAL`](../../Reference/3dmet/M3dmetControl.md).
  > **Note:** The Manhattan distance is the combined distance in all 3 axes between two positions, as opposed to the length of the shortest path between the two positions. For example, the shortest distance from (0,0,0) to (1,1,1) is ~1.73, but the Manhattan distance is 3 (1 + 1 + 1). Typically, you should not use [`M_MANHATTAN_DISTANCE_TO_SURFACE`](../../Reference/3dmet/M3dmetStat.md)unless it is specifically required for your application.
- For a cylinder 3D geometry object, you can choose to measure distances from each source point to the nearest position on the central axis of the cylinder. The central axis is extended infinitely, even if the cylinder is finite. To calculate these distances using [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md) or[`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md), set[`DistanceType`](../../Reference/3dmet/M3dmetStat.md) to [`M_DISTANCE_TO_CENTER_AXIS`](../../Reference/3dmet/M3dmetStat.md)or [`M_DISTANCE_TO_CENTER_AXIS_SQUARED`](../../Reference/3dmet/M3dmetStat.md). For [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), use[`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_PAIRING_TYPE`](../../Reference/3dmet/M3dmetControl.md) set to [`M_DISTANCE_TO_REFERENCE`](../../Reference/3dmet/M3dmetControl.md), and [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_DISTANCE_METRIC_OPERATOR`](../../Reference/3dmet/M3dmetControl.md) set to [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SQUARED`](../../Reference/3dmet/M3dmetControl.md) for the same result.
- For a line 3D geometry object, distances are measured from each source point to the nearest position on the line. Unlike the central axis of a cylinder, a finite line is not extended infinitely for this calculation. To calculate these distances using [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md) or[`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md), set[`DistanceType`](../../Reference/3dmet/M3dmetStat.md) to[`M_DISTANCE_TO_LINE`](../../Reference/3dmet/M3dmetStat.md)or [`M_DISTANCE_TO_LINE_SQUARED`](../../Reference/3dmet/M3dmetStat.md). For [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), use [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_PAIRING_TYPE`](../../Reference/3dmet/M3dmetControl.md) set to [`M_DISTANCE_TO_REFERENCE`](../../Reference/3dmet/M3dmetControl.md), and use [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_DISTANCE_METRIC_OPERATOR`](../../Reference/3dmet/M3dmetControl.md) set to [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SQUARED`](../../Reference/3dmet/M3dmetControl.md) for the same result.
- For a plane 3D geometry object, you can choose to measure distances from each source point to the position on the surface of the plane that is directly above or below the point along the Z-axis. To calculate these distances using [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md) or[`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md), set[`DistanceType`](../../Reference/3dmet/M3dmetStat.md) to [`M_ABSOLUTE_DISTANCE_Z_TO_SURFACE`](../../Reference/3dmet/M3dmetStat.md)or [`M_SIGNED_DISTANCE_Z_TO_SURFACE`](../../Reference/3dmet/M3dmetStat.md). For [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), use [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_PAIRING_TYPE`](../../Reference/3dmet/M3dmetControl.md) set to [`M_DISTANCE_TO_SURFACE`](../../Reference/3dmet/M3dmetControl.md), and use [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_DISTANCE_METRIC_OPERATOR`](../../Reference/3dmet/M3dmetControl.md) set to [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SIGNED`](../../Reference/3dmet/M3dmetControl.md) for the same result.
- For a sphere 3D geometry object, you can choose to measure distances from each source to the center of the sphere. To calculate these distances using [`M3dmetStat`](../../Reference/3dmet/M3dmetStat.md) or[`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md), set[`DistanceType`](../../Reference/3dmet/M3dmetStat.md) to[`M_DISTANCE_TO_CENTER`](../../Reference/3dmet/M3dmetStat.md)or [`M_DISTANCE_TO_CENTER_SQUARED`](../../Reference/3dmet/M3dmetStat.md). For [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), use [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_PAIRING_TYPE`](../../Reference/3dmet/M3dmetControl.md) set to [`M_DISTANCE_TO_REFERENCE`](../../Reference/3dmet/M3dmetControl.md), and use [`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with [`M_DISTANCE_METRIC_OPERATOR`](../../Reference/3dmet/M3dmetControl.md) set to [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SIGNED`](../../Reference/3dmet/M3dmetControl.md) for the same result.

When calculating distances to the surface of a 3D geometry object, you can specify to have distances returned as either absolute or signed values (for example, using [`M_ABSOLUTE_DISTANCE_TO_SURFACE`](../../Reference/3dmet/M3dmetStat.md) or [`M_SIGNED_DISTANCE_TO_SURFACE`](../../Reference/3dmet/M3dmetStat.md) with [`M3dmetDistance`](../../Reference/3dmet/M3dmetDistance.md), respectively; for [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), use[`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with[`M_DISTANCE_METRIC_OPERATOR`](../../Reference/3dmet/M3dmetControl.md) set to [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SIGNED`](../../Reference/3dmet/M3dmetControl.md) for the same result). When you specify to return signed values, the sign indicates whether the point is above or below the surface of the reference object.

Some types of distance calculations with 3D geometries (such as [`M_DISTANCE_TO_CENTER`](../../Reference/3dmet/M3dmetStat.md)) include a square root operation to determine the distance for each point in the source. You can improve the performance of these types of distance calculations by specifying an alternate version (such as, [`M_DISTANCE_TO_CENTER_SQUARED`](../../Reference/3dmet/M3dmetStat.md); for [`M3dmetDistanceEx`](../../Reference/3dmet/M3dmetDistanceEx.md), use[`M3dmetControl`](../../Reference/3dmet/M3dmetControl.md) with[`M_DISTANCE_METRIC_OPERATOR`](../../Reference/3dmet/M3dmetControl.md) set to [`M_ABSOLUTE`](../../Reference/3dmet/M3dmetControl.md) or [`M_SIGNED`](../../Reference/3dmet/M3dmetControl.md) for the same result) that bypasses this step and instead stores the square of the distance in the result buffer.
