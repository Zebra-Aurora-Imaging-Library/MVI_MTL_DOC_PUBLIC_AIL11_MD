# Aurora Imaging Example Launcher

Aurora Imaging Example Launcher, which is accessible from Aurora Imaging Control Center, lets you run all of the examples that the library has to offer. Examples are easy to find because the interface organizes them intuitively, based on their general functionality. For instance, if you are interested in a specific aspect of the library, such as its 3D, deep learning, or OCR capabilities, you can easily locate and run these examples.

*[Image: sc_ExampleLauncher.png]*

The interface also lets you open an example in several different programming languages and environments, so you can view, edit, and recompile the code. Examples not only show you what the library can do, but can be the base from which you write your own machine vision application.
