# Aurora Imaging Intellicam

Aurora Imaging Intellicam is an image capture and frame grabber configuration utility that lets you interactively configure camera, system, and digitizer settings. For example, you can use Aurora Imaging Intellicam's Feature Browser to test I/O signal settings (see [Using Aurora Imaging Intellicam's Feature Browser](../C56_IO_signals/S02_Steps_to_use_IO_signals.md)). You can also use it to create and/or modify digitizer configuration format (DCF) files. Aurora Imaging Intellicam is useful when working with Camera Link products.

You can launch Aurora Imaging Intellicam from the Aurora Imaging Control Center.
