# Aurora Imaging Configurator

Aurora Imaging Configurator (_AILConfig.exe_), launched through Aurora Imaging Control Center, provides licensing, reserved non-paged memory configuration, and system information tools. For example, if you need to change the amount of reserved non-paged memory for the library, you can do so at any time using Aurora Imaging Configurator. For more information on memory management options, see [Advanced memory management](../C23_Data_buffers/S18_Advanced_memory_management.md).

Should you require technical support, use Aurora Imaging Configurator's **Troubleshooting** page to generate a text file that contains all the necessary system information required for basic troubleshooting; this file can then be forwarded to your Zebra technical support representative. For more information on this text file, see [Interactive log](../C65_Development_and_Debugging_Tools_and_Techniques/S01_Profiler.md).
