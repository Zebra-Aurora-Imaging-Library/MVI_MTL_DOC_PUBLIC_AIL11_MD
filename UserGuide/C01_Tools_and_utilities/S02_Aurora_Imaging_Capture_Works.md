# Aurora Imaging Capture Works

Aurora Imaging Capture Works, which is accessible from Aurora Imaging Control Center, is a GUI for 2D and 3D GenICam-compliant imaging devices that use standard interfaces like GigE Vision, USB Vision, CoaXPress, or GenTL. With this GUI, you can rapidly evaluate the performance and functionality of almost any GenICam-compliant camera or 3D sensor (or other device). You can also allocate your device, start or stop acquisitions, analyze and visualize captured data, save grabbed data, send a software trigger, as well as browse and control the selected device's features (for example, by using the Feature Browser).

You can find more information in the Aurora Imaging Capture Works user guide, available from the **Help** menu item within the utility. You can also access this user guide online at [zebra.com/ail-docs](https://zebra.com/ail-docs).
