# Aurora Imaging Profiler

Aurora Imaging Profiler which is accessible from Aurora Imaging Control Center, is a utility that creates and graphically displays trace logs that can contain information such as when each Aurora Imaging Library function call in each thread starts and stops. These logs and the information therein can help you to analyze your application's execution for debugging or optimization.

For more information, see [Aurora Imaging Profiler and trace logs](../C65_Development_and_Debugging_Tools_and_Techniques/S01_Profiler.md).
