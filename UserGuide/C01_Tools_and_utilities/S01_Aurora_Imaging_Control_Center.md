# Aurora Imaging Control Center

Aurora Imaging Control Center is a user interface that serves as the library's main hub, providing access to all important applications and files. For example, you can access Aurora Imaging Configurator, Aurora Imaging Example Launcher, Aurora Imaging CoPilot, Aurora Imaging Profiler, release notes, interactive utilities related to processing and analysis modules, and many other useful files such as predefined contexts and example images.

Aurora Imaging Control Center also lets you launch Aurora Imaging Intellicam, which can provide camera interfacing and interactive access to grab features and functionalities of your frame grabber and otherwise compliant cameras. You can also use Intellicam to create and/or modify digitizer configuration format (DCF) files. Intellicam has an interactive tool called Feature Browser which can let you dynamically configure camera, system, and digitizer settings.

If you have more than one version of the library installed on your computer, all versions will appear in Aurora Imaging Control Center. You must use the Control Center to select the version that you want to use, and you can toggle between versions to access the required applications and files. See [Side-by-side installations](../C02_Building_an_application/S03_Going_from_AIL_10.7_to_AIL_11.md) for more information.
