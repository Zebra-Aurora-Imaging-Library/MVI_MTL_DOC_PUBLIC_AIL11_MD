# im

> The functions prefixed with Mim make up the Image Processing module. The basic Image Processing module allows the user to perform point-to-point, statistical, spatial filtering, morphological, wavelet, and geometric transformation operations. These operations allow you to perform image enhancements, distortion corrections, and analyses.
