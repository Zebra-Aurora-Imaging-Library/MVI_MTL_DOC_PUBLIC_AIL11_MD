# reg

> The functions prefixed with Mreg make up the Registration module. The Registration module allows you to find the transformations that optimally position images in the same coordinate system, to mosaic images, to extend your depth of field, to generate an index map from a scene, or to detect defects on a reflective or smooth surface using photometric stereo. The Aurora Imaging Library Registration module uses multiple images of the same, or of overlapping scenes, to generate an image.
