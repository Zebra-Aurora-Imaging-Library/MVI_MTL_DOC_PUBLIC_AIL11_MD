# 3dim

> The functions prefixed with M3dim are responsible for 3D image processing containers that hold 3D data. The main components of a container ([`M_CONTAINER`](../../Reference/buf/MbufControlContainer.md)) are [`M_COMPONENT_RANGE`](../../Reference/buf/MbufControlContainer.md) and [`M_COMPONENT_CONFIDENCE`](../../Reference/buf/MbufControlContainer.md). Optional components are [`M_COMPONENT_REFLECTANCE`](../../Reference/buf/MbufControlContainer.md), [`M_COMPONENT_INTENSITY`](../../Reference/buf/MbufControlContainer.md), [`M_COMPONENT_NORMALS_AIL`](../../Reference/buf/MbufControlContainer.md) and [`M_COMPONENT_MESH_AIL`](../../Reference/buf/MbufControlContainer.md).
