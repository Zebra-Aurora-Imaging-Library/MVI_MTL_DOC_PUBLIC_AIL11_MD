> Get the label value of a blob at a specified position.

## Syntax

```c
AIL_INT MblobGetLabel(
    AIL_ID    ResultBlobId,  //in
    AIL_INT   XPos,          //in
    AIL_INT   YPos,          //in
    AIL_INT * LabelVarPtr    //out
)
```

## Description

This function gets the label value of a specified blob. Label values are used, for example, to obtain calculation results for single blobs. Blob label values must have been generated by calling [`MblobCalculate`](../../Reference/blob/MblobCalculate.md).

## Parameters

### `ResultBlobId` *(in, AIL_ID)*

Specifies the identifier of the blob analysis result buffer where the labels are stored. Note, this function cannot obtain the label values of blobs that have been deleted from the result buffer with [`MblobSelect`](../../Reference/blob/MblobSelect.md).

### `XPos` *(in, AIL_INT)*

Specifies the X-coordinate of the blob.

### `YPos` *(in, AIL_INT)*

Specifies the Y-coordinate of the blob.

### `LabelVarPtr` *(out, *AIL_INT)*

Specifies the address of the variable in which to write the label value. If there is no blob at the specified location, the blob has been deleted, or if the blob's [`XPos`](../../Reference/blob/MblobGetLabel.md) and [`YPos`](../../Reference/blob/MblobGetLabel.md) lie outside the original identifier image, [`M_NULL`](../../Reference/blob/MblobGetLabel.md) is returned instead of the label value. Also, since [`MblobGetLabel`](../../Reference/blob/MblobGetLabel.md) returns the label value, you can set this parameter to `M_NULL`.

## Return Value

**Type:** `AIL_INT`

The returned value is the label value of the specified blob.
