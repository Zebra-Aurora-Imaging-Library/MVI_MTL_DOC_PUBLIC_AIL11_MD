# thr

> The functions prefixed with Mthr make up the Thread module. The Thread module supports allocation of Aurora Imaging Library thread contexts and synchronization events. Using the functions of the Thread module, you can also control the created Aurora Imaging Library thread contexts and events, inquire about various settings, and synchronize execution of multiple threads.
