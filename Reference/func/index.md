# func

> The functions prefixed with Mfunc make up the Function Development module. The Function Development module allows programmers to define functions to extend Aurora Imaging Library's functionality. Using this toolkit, you can implement functions and integrate them directly into Aurora Imaging Library, where they behave like standard Aurora Imaging Library functions (for example, respecting error handling and tracing).
