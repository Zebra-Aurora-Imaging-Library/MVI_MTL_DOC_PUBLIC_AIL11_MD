# obj

> The functions prefixed with Mobj make up the Object module. The Object module allows you to create Aurora Imaging Library utility objects, as well as to control and inquire Aurora Imaging Library utility objects and other Aurora Imaging Library objects, for a Distributed Aurora Imaging Library publishing application.
