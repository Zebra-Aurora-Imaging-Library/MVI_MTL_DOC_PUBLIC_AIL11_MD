# col

> The functions prefixed with Mcol make up the Color Analysis module. The Color Analysis module allows you to perform color-based operations such as color transformation using relative color calibration, matching, projection, and distance. You can use these operations to quickly design powerful applications that can correct colors, identify colors, segment colors (supervised), detect defects, enhance color-to-grayscale conversion, separate superimposed colors, and measure the relative presence of a color in an image.
