# sys

> The functions prefixed with Msys make up the System module. The System module supports the allocation and inquiry of systems. The system control represents a physical board, most commonly an imaging frame grabber. The system control also allows you to access the graphics controller and Host CPU. You can use the system control to specify the physical device to access, and to set some system-wide settings. Once the system control is configured, you can add other components to your application, such as a digitizer control, to control specific aspects of the device.
